# HTML 5 + Bootstrap 3 + AngularJS 1.5 Development Template

## Getting Started
* npm install
* node_modules/bower/bin/bower install
* ln -s ../bower_components ./src/libs
* gulp

## Commands
* gulp
* gulp sass
* gulp jshint
* gulp watch
* gulp build
* gulp publish:aws --bucket BUCKET_NAME --key ACCESS_KEY_ID  --secret SECRET_ACCESS_KEY
